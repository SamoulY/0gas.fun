# GasFree Frontend Demo

React + Vite example showing the full user flow: connect wallet, answer question, watch ad, receive reward.

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and set your.
3. `npm run dev`
4. Open `http://localhost:5173`

## Environment Variables
Create `.env` with:
```
VITE_API_BASE_URL=http://localhost:3000
```
